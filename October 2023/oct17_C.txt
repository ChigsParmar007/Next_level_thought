1. Jira software: 
  Jira software ma aapne badha devlopers mate new new task banavi sakiae chhiae
  and aek bija ne assign kari sakiae.
  jira ma aapne sprint planning pn kari sakiae chhiae aema. sprint means ke aaone next thodak divso ma shu shu kam 
  karshu kai rite karshu kon karse ae badhu manage kari sakiae chhiae mostly 1 sprint 15 divs ni hoy che
  jira use karvathi ghana benefits rey che main to kam vechai jaay che tenathi devlopment ni process zadpi thai jay che

2. Bitbucket
  Bitbucket github ni jevu j che aema aapne code ne manage kari sakiae chhiae 
  Bitbucket ma git na j commnad use thai che
  jyare koi aek devloper koi PR banave toh pela ae seniors check kari sake and jo ae perface hoy to j ae code staging or production ma 
  merge kare jenathi error aavana chances ochha thai jay che and jo PR ma koi problem hoy to aema commnnt karine kai sakiae ke ahiya change karvano che
  Bitbucket jyare 2 devlopers alg alg jgya parthi kam karta hoy tyre satuhi best che
  Bitbucket ma aapne pipeline pn set kari sakiae chhiae like aapne code merge kari didho pachhi ae code automatic built thavanu start thai jay.
